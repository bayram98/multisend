# Milti Send 


## Installation

```
npm install
```

## Prerequisites

Create `.env` from `.env.example`

```shell
cp .env.example .env
```

and fill in environment variables.

## Run locally

### Start local hardhat node
```shell
    npx hardhat node --hostname 127.0.0.1
```

### Compile and run test
```shell
    npx hardhat compile
    npx hardhat test
```
